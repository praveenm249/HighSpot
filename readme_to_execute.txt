Steps to run the code:
----------------------
0. Install Python 3.x (if not installed)
1. Modify changes.json, if you want to change the test case, in files directory under the project directory (HighSpot).
2. Open terminal and change directory to this project directory (HighSpot).
3. Enter the command `chmod 755 batchscript.sh` to change the permissions on the file.
4. Enter the command `./batchscript.sh`
5. Verify the changes in output.json in the files directory.
